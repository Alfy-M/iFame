# iFame

## What's it?
An Android  application for booking meals from  menu of registred resturants.

## Application Modules

### User Module
* **Registration**
* **Login**
* **Booking**

### Caterer Module
**Registration**
**Login**
**Restaurant Insertion**

### Client

### Server (Computing, Authentication and Database)
